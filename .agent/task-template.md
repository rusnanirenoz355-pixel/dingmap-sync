## 任务卡自查

1. 是否只做本任务范围内的内容。
2. 是否明确写了不做范围。
3. 是否包含 GitHub Issue 同步要求。
4. 是否包含跨设备继续开发要求。
5. 是否包含 docs/dev-log.md 更新要求。
6. 是否包含命令验证清单。
7. 是否避免提交敏感文件。
8. 是否统一写 pnpm run setup，而不是 pnpm setup。
9. 是否明确钉图一键录入优先，Playwright 逐条录入备用。
10. 是否判断是否需要更新 README。
11. 是否判断是否需要更新 .agent 规则。
12. 是否输出完成报告。
