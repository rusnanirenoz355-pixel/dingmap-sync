# 需求摘要

本项目是“钉图自动化同步工作台”，依据《钉图自动化同步工作台_需求大纲_V2.docx》和任务卡 001 创建第一版项目地基。

## 核心目标

1. 支持优招、捷聘、后续网页、手动粘贴、Excel 等多来源数据。
2. 所有来源数据先进入 Raw Table，保留原始抓取或导入内容。
3. 清洗后的数据进入 Clean Table，作为真实可录入钉图的数据。
4. 第一优先级是 Clean Table 导出钉图一键录入模板 / 数据包。
5. Playwright 逐条填写钉图仅作为备用方案。
6. 新增来源必须走 DataSourcePlugin 插件接口。
7. 必须支持粘贴模板信息自动进入 Clean Table。
8. 必须支持跨电脑继续开发。

## 第一版范围

本任务只创建基础工程、数据表、类型、插件接口、占位页面、规则包、文档和 CI，不做真实采集、真实登录、真实录入和复杂同步逻辑。

## 数据流

DataSourcePlugin -> Raw Table -> Normalizer -> Clean Table -> Sync Plan -> DingMap 一键录入导出。
