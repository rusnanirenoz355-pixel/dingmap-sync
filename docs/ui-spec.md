# OKX 浅色 UI 规范

## 颜色

| 用途 | 色值 |
| --- | --- |
| 页面背景 | #F5F6F7 |
| 卡片背景 | #FFFFFF |
| 表头背景 | #F2F3F5 |
| 主文字 | #0B0B0F |
| 次文字 | #5F6368 |
| 弱文字 | #9CA3AF |
| 边框 | #E5E7EB |
| 主按钮 | 黑底白字 |
| 成功 | 绿色 |
| 失败 | 红色 |
| 待确认 | 黄色 |
| 更新 | 蓝色 |

## 组件原则

1. 使用浅灰页面背景和白色内容面。
2. 表格、卡片、按钮保持 8px 以内圆角。
3. 主操作按钮使用黑底白字。
4. 状态标签使用清晰状态色，不使用复杂动画。
5. 页面第一屏就是工作台，不做营销落地页。
